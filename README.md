OrganizePro To-Do List Application
Overview

OrganizePro is a modern and user-friendly To-Do List web application designed to help users manage daily tasks efficiently. The application allows users to create, edit, prioritize, and track tasks with due dates. All data is stored locally in the browser using localStorage, making it fast and completely offline.

Features

Add, edit, and delete tasks

Mark tasks as completed or active

Set task priority (High, Medium, Low)

Assign due dates to tasks

Filter tasks by All, Active, Completed, Due Today, and Overdue

Search tasks by title or tags

Drag and drop to reorder tasks

Dark and Light theme toggle

Export tasks to JSON file

Import tasks from JSON file

Responsive design for desktop and mobile devices

Offline support using browser localStorage

Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

Browser localStorage

Project Structure

This project is built as a single HTML file that contains:

HTML for structure

CSS for styling and themes

JavaScript for functionality and data management

How to Use

Download or clone the repository

Open the ToDo_LIST.html file in any modern web browser

Start adding tasks using the input field

Set due date and priority as needed

Use filters and search to manage tasks efficiently

Toggle between dark and light themes

Export or import tasks using JSON files

Data Storage

All tasks are stored locally in the browser using localStorage. No external database or server is required.

Use Cases

Daily task management
